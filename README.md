# Verificador-de-N-mero-Par-ou-mpar
## Como funciona O usuário digita um número e o programa informa se ele é par ou ímpar.   Depois, o sistema pergunta se deseja testar outro número.  ## Objetivo Este projeto foi criado para praticar: - Entrada de dados - Conversão de valores - Estruturas condicionais - Estruturas de repetição - Lógica de programação
