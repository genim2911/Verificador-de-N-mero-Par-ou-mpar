# Verificador de Numeros par ou impar

Este é um projeto simples em JavaScript que veriica se um numero digitado pelo usuario é **par** ou **impar**.

## Tecnologias usadas
- HTML
- JavaScrpit

## Como funsiona 
O usuário digita um número e o programa informa se ele é par ou ímpar.  
Depois, o sistema pergunta se deseja testar outro número.

## Objetivo
Este projeto foi criado para praticar:
- Entrada de dados
- Conversão de valores
- Estruturas condicionais
- Estruturas de repetição
- Lógica de programação

## Como executar
1. Baixe os arquivos
2. Abra o arquivo `index.html` no navegador
3. Siga as instruções que aparecem na tela
